# 闭包及优缺点

# 定义

  • 有权访问另一个函数作用域中变量的函数

# 目的

  • 为了设计私有变量和方法

# 优点

  • 避免全局变量的污染

# 缺点

  • 闭包会常驻内存,增大内存的使用量(函数中的变量都被保存在内存中)
